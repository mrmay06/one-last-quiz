# SFX placeholder

Add free SFX from Pixabay/Freesound here:
- `stinger.mp3`        — channel signature, 0.3s, plays at hook + cliffhanger
- `ding.mp3`           — iMessage notification, used in `imessage` sub-format
- `whoosh.mp3`         — image cut transitions
- `dispatch_radio.mp3` — radio static for `dispatch` sub-format
- `typewriter.mp3`     — for `note` sub-format

These are optional; templates will render without them. Wire them up in templates after adding files.
